# Olympics-Tableau-Dashboard
📊 Olympics Data Analysis – Tableau Dashboard
📌 Project Overview

This project presents an interactive Tableau dashboard analyzing historical Olympic data to uncover meaningful insights about country performance, gender participation, and athlete trends.

🎯 Objectives

Analyze country-wise medal performance

Study gender participation trends

Explore athlete age distribution

Visualize participation growth over years

🛠 Tools Used

Tableau Public

CSV Dataset

Data Cleaning & Transformation
